# Camp Rating System

**Camp Rating** е уеб приложение, което помага на потребители да намират, оценяват и рецензират места за къмпингуване.

## Видове потребители

- **Администратор**
  - Управлява потребители (редакция, изтриване – без добавяне на нови администратори)
  - Управлява места за къмпинг (CRUD)
  - Преглежда ревюта
  - Вижда Dashboard със статистики

- **Обикновен потребител**
  - Разглежда и търси места за къмпинг по име
  - Добавя ревюта към места
  - Може да редактира и изтрива само **своите ревюта**
  - Разглежда "Моите ревюта"

---

## Основни функционалности

### Автентикация и роли
- ASP.NET Identity
- Роли: `Admin`, `User`
- Потребителите имат: Име, Фамилия, Имейл, Парола, Роля

### Къмпинг места
- Име (до 64 символа)
- Описание (до 255 символа)
- Географска ширина и дължина (double)
- Снимка (до 2MB)
- CRUD за админ
- Публичен списък + търсачка по име

### Ревюта
- Потребителите могат да добавят, редактират и трият **само своите ревюта**
- Списък „Моите ревюта“

### Dashboard (Admin)
- Брой потребители
- Брой места
- Брой ревюта
